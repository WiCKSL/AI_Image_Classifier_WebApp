How the Model Reached This Result

The detection model is based on the MobileNetV2 architecture, a convolutional neural network optimized for efficient image feature extraction. The uploaded image is first resized and normalized before being processed by the pretrained convolutional layers. These layers identify visual features such as patterns, edges, and structural inconsistencies that commonly appear in synthetic AI generated images. After feature extraction, the fully connected layers perform binary classification to determine whether the image belongs to the category of real photographs or AI generated images.

Model Architecture

MobileNetV2

A lightweight convolutional neural network designed for efficient image recognition tasks.

Transfer Learning

Pretrained weights from ImageNet allow the model to recognize complex visual patterns with limited training data.

Feature Extraction

The convolutional layers analyze textures, shapes, and visual artifacts within the image.

Binary Classification

The final dense layer predicts whether the image is real or AI generated using sigmoid activation.

Project Overview

The AI Image Authenticity Detection System is developed as part of the MSc Computer Science final year project. The objective of this project is to explore how deep learning models can detect synthetic images generated by modern AI tools. By leveraging transfer learning and convolutional neural networks, the system provides a practical demonstration of how artificial intelligence can be used to identify manipulated visual content.
